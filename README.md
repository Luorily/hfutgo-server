因为ThinkPHP实在是太冗杂了，而且PHP实在是不适合做爬虫，故决定重置并开源，目前正在开发中。

- 洗衣机查询（企鹅洗衣、海尔云洗衣、美的U净洗衣）
- 新门户逻辑处理
- 第二课堂查询
